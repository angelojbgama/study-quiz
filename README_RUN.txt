Como rodar:
1) npm install
2) npm start
3) Abra no Android com Expo Go ou emulador
3) Build npx --yes eas-cli@latest build --platform android --profile preview
Importe CSV (quiz,pergunta,resposta,explicacao,tags) ou JSON ([{quiz,question,answer,explanation?,tags?}]).
Filtros por múltiplas tags e modo SRS (Somente vencidos) estão nas telas Cartões e Aprender.
